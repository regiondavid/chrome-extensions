### 扇贝查词助手
#### 范围
[https://www.theguardian.com](https://www.theguardian.com) 下的所有新闻页面
#### 功能
- 划词翻译功能。包括基本的解释、音标和发音。
- 去广告功能。
- 分页功能。点击右侧两个按钮即可进行上下翻页。
